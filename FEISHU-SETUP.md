## 📋 快速接入步骤（5分钟）

### 第1步：创建飞书机器人（2分钟）

1. 打开飞书APP或网页版
2. 找一个群聊（或创建新群）
3. 点击群设置 → 机器人 → 添加机器人
4. 选择"自定义机器人"
5. 设置名称：**AI学习导师**（可自定义）
6. 设置描述：**70天AI全栈学习导师系统**
7. 点击"添加"
8. **复制webhook地址**（重要！）

webhook地址格式：
```
https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

---

### 第2步：配置webhook地址（1分钟）

打开 `D:\demo\ai-techer\.env.local` 文件，修改第2行：

```bash
# 飞书机器人配置
FEISHU_WEBHOOK_URL=https://open.feishu.cn/open-apis/bot/v2/hook/你的webhook地址

# 其他配置（暂时可以忽略）
# DATABASE_URL=postgresql://...
# REDIS_URL=redis://...
# OPENAI_API_KEY=sk-...
```

把`你的webhook地址`替换成刚才复制的webhook地址。

---

### 第3步：测试飞书推送（2分钟）

运行测试脚本：

```bash
cd D:\demo\ai-techer

# 安装依赖
npm install

```

创建测试文件 `test-feishu.ts`：

运行测试：
```bash
npx tsx test-feishu.ts
```

如果成功，你会在飞书群里看到：


### 2. 添加@提醒

如果想@特定人，修改消息添加：

```typescript
{
  tag: 'at',
  user_id: '你的飞书user_id'
```

如果想推送到多个群，修改 `.env.local`：

```bash
FEISHU_WEBHOOK_URL_1=https://...  # 学习群
FEISHU_WEBHOOK_URL_2=https://...  # 打卡群
FEISHU_WEBHOOK_URL_3=https://...  # 朋友监督群
```

## ❓ 常见问题

### Q1: webhook地址在哪里找？
A: 飞书群 → 设置 → 机器人 → 自定义机器人 → 复制webhook地址

### Q2: 推送失败怎么办？
A: 检查：
1. webhook地址是否正确
2. 机器人是否被移除
3. 网络是否正常
4. 运行 `npx tsx test-feishu.ts` 查看错误

### Q3: 可以推送到个人吗？
A: 不行，飞书机器人只能推送到群聊。建议创建一个"AI学习日记"群，就你一个人。

### Q4: 如何暂停推送？
A: 两种方式：
1. 临时：把 `.env.local` 中的 `FEISHU_WEBHOOK_URL` 注释掉
2. 永久：飞书群里移除机器人

### Q5: 消息太多了，可以调整吗？
A: 可以，修改定时任务的推送逻辑，改为每周推送或只推送重要节点。
