# Agent 全栈开发工程师 60 天学习计划

## 📋 总览

这是一份为期 60 天的 AI Agent 全栈开发工程师学习计划，通过 4 个递进项目，从前端工程师转型为 Agent 全栈工程师。

## 🎯 学习路径

```
Day 1-180（前 6 个月）：Node 全栈 + TS Agent
Day 181-365（第 7-12 个月）：TS Agent 深度
Day 366-540（第 13-18 个月）：Python 补齐 + 双栈
```

本计划聚焦前 60 天（第一阶段的前 1/3）。

## 🏗 4 个递进项目

### P1：智能异常监控平台（Day 1-15）
- **核心**：单 Agent + Tool Calling + 实时监控
- **技术栈**：Vue 3 + NestJS + Postgres + WebSocket + DeepSeek
- **目标**：建立后端思维 + AI 集成基础
- **简历定位**：Node 全栈 + AI 应用入门

### P2：故障知识库 RAG Agent（Day 16-30）
- **核心**：RAG 完整链路 + LangChain.js
- **技术栈**：P1 基础 + pgvector + LangChain.js + Embedding
- **目标**：掌握 RAG 工程化
- **简历定位**：RAG 应用开发工程师

### P3：多 Agent 运维系统（Day 31-45）
- **核心**：LangGraph.js + Multi-Agent 协作
- **技术栈**：P2 基础 + LangGraph.js + Eval 体系
- **目标**：掌握 Agent 编排
- **简历定位**：Agent 全栈工程师（TS）

### P4：3D 智能机房平台（Day 46-60）
- **核心**：生产级 Agent + Three.js 差异化
- **技术栈**：P3 基础 + Three.js + 多模态 + 监控
- **目标**：综合上线 + 真实用户
- **简历定位**：资深 Agent 应用工程师

## 🎓 学习方式

### 每日节奏（工作日 2h + 周末 6h）
```
工作日（周一到周五）：
  - 学习材料：20-30 分钟
  - 编码实践：90-100 分钟
  - 产出验收：按清单打勾

周末（周六日）：
  - 整块时间：6 小时
  - 深度项目推进
  - 复盘 + 下周预习
```

### 学习原则
1. **项目驱动**：不为学而学，缺啥补啥
2. **及时验收**：每天必须有可验收产出
3. **AI 编程加速**：用 Claude Code 辅助，但核心逻辑自己写
4. **真实场景**：所有项目对标工业真实需求
5. **持续积累**：每个项目都是下一个的基础
