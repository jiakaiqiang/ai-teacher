## 💡 核心原则

### 1. 理解 > 完成

```
❌ 错误：今天完成 Day 5 所有任务
✅ 正确：今天彻底理解 Tool Calling 原理
```

### 2. 复制代码 ≠ 学会编程

```
AI 生成 500 行代码 → 你复制粘贴 → 学到：0%
你手写 500 行代码 → 花 2 小时 → 学到：100%
```

### 3. 学习是反人性的

```
人性：想快速完成 → 复制 AI 代码
现实：快速完成 = 没有学习
解决：强制自己手写核心代码
```
