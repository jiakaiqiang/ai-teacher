## 🎨 示例对话

### 示例1：遇到技术问题
```
你："我不知道Next.js的Server Components怎么用"

AI导师：
"Server Components是Next.js 13引入的新特性，让我给你详细讲解：

【核心概念】
- 在服务器端渲染，减少客户端JS
- 可以直接访问数据库
- 默认所有组件都是Server Component

【使用方法】
1. 创建组件文件（不需要'use client'）
2. 可以直接用async/await
3. 示例代码：

```tsx
// app/posts/page.tsx
async function getPosts() {
  const res = await fetch('https://api.example.com/posts')
  return res.json()
}

export default async function Posts() {
  const posts = await getPosts()
  
  return (
    <div>
      {posts.map(post => (
        <div key={post.id}>{post.title}</div>
      ))}
    </div>
  )
}
```

【注意事项】
- 不能使用useState、useEffect等hooks
- 不能添加事件监听器
- 需要交互的部分用'use client'

【实战建议】
在lighthouse-ai-analyzer项目中：
- 数据获取组件用Server Components
- 交互按钮用Client Components

需要我给你写一个具体的例子吗？"
```
